Privacy Policy

This Privacy Policy applies to the FSK Comm website and related software materials provided by BeepSystems.

The project website is informational. BeepSystems does not require account registration to access it and does not use the site to collect personal or sensitive user data for product operation.

Information We Collect

BeepSystems does not ask visitors to create accounts or submit personal data to use this site.

  • No account is required to access the public website.
  • No in-product personal profile is created through this site.
  • No advertising or data brokerage is performed through this site.

Software and Local Data

The FSK Comm project is intended to operate on the user's own devices and local integrations. Message handling, audio capture, keys, and diagnostics are controlled by the host application or local environment, not by this website.

Data Retention

Because this site does not provide accounts or backend message storage, BeepSystems does not retain server-side personal message content through the public project website.

Security

The project itself supports authenticated encryption for message payloads, but visitors should evaluate their own host application, device, and deployment security separately.

Children's Privacy

This site is not directed to children under 13, and BeepSystems does not knowingly collect personal information from children through it.

Changes

BeepSystems may update this policy if the website or related product practices change. Updates will be published on this page.

Contact

Questions about this policy can be sent to contact@beep.systems.
